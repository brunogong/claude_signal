# 📊 Forex Analysis Pro

Analisi completa delle coppie forex con:
- ✅ Analisi Tecnica (RSI, MACD, Bollinger, EMA, ADX, Ichimoku)
- ✅ Analisi Fondamentale (Tassi interesse, Calendario economico)
- ✅ Analisi Sentiment (News, Retail positioning, COT)
